# Smart Cane for the Visually Impaired

Final-year Electrical Engineering project focused on assisting visually impaired users by detecting obstacles and stairs using sensors and machine learning.

## Overview

The system is designed to help visually impaired users navigate their environment safely by detecting obstacles and providing audio alerts.

## Features

- Obstacle detection using ultrasonic sensors
- Stair detection using a camera and a machine learning model to detect stairs
- Real-time audio alerts to the user
- SOS message to caretaker using telegram
- Embedded sensing system

## System Architecture

The system integrates sensors, processing hardware, and audio feedback to warn the user about obstacles.
